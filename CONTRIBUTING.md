# Contributing to LinkedIn Detox

Thank you for your interest in contributing to LinkedIn Detox!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b my-feature`)
3. Make your changes
4. Submit a pull request

## Guidelines

- Keep it vanilla JS — no build tools, no frameworks
- Test your changes with `npm test` (vitest)
- Make sure the extension still loads as an unpacked extension in Chrome

## Questions?

If you have questions, open an issue or contact the maintainer before contributing.
